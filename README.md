## Current Projects

### Sicilian Language Model Project

I'm partnering with Camdemia Siciliana to create an open weight LLM that is fluent in Sicilian. This project is part of my MSCS thesis work at Georgia Tech and will be fully open source. The model weights, training datasets, and training methods will all be publicly available.

My research focuses on training LLMs with sparse datasets, which is a critical challenge for low resource languages. I'm specifically working on identifying and recognizing culturally loaded terms in and between languages to enable more fluent translations. This is especially important for underrepresented languages where existing translation systems often miss cultural nuance.
